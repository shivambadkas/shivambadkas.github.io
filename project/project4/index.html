Ridge and Lasso Regression

Predicting Insurance charges with regression

Link to Project

A common problem that data scientists face is overfitting - their model follows the noise too closely. Regularization is basically a technique that adds a penalty for feature coefficients with large values, L1 and L2 are utilized by lasso and ridge regression respectively. In this project I use a dataset of insurance charges for healthcare expenses of households measure by predictors such as location, number of children, smoking status, and age. I then use linear regression to predict charges and try to see if ridge and lasso regression can create a better fit for this model.

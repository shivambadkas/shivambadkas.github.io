Dimensionality Reduction

PCA t-SNE

Link to Project

One major problem of high dimensional data is the difficulty in interpreting and computing data not in three dimensions. The curse of dimensionality refers to the problems that arise with data with a high amount of features, therefore it is wise to reduce dimensions for many reasons including computation, model performance, and interpretability. Principal Components Analysis is a linear unsupervised learning algorithm which essentially transforms the original set of vectors into a new set. The new set preserves global structure of the data at the cost of local structure. T-distributed Stochastic Neighbor Embedding is a non-linear unsupervised learning algorithm which basically embeds points from a high dimension to a low dimension by preserving the neighborhood of that point. Therefore, unlike PCA, it preserves local structure of data. In this project I use PCA and t-SNE on the MNIST dataset, a collection of hand written digits with each pixel being a feature, 42000x785, with one column represeting what number the image represents. I use PCA and t-SNE to see the differences in strengths between the two algorithms.
